# 引用映射表

本文件用于记录 Word 草稿中的手写引用编号与 LaTeX BibTeX key 的对应关系。

## 规则

- Word 原编号 `[1]`、`[2]` 等不直接保留在最终论文中。
- LaTeX 正文中统一使用 `\cite{}`。
- 参考文献统一维护在 `refs/references.bib`。
- 引用编号由 biblatex 自动生成。
- 多文献引用使用 `\cite{key1,key2}`。
- 标题中不使用引用标注。
